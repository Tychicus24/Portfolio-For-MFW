This is a logo I made for a my Geekwise Bootcamp 1 Basic HTML and CSS final class project. I was originally using the Geekwise Owl logo and when I asked Geekwise if it was OK to use the logo in mky project I was told that it could not be used but that I was welcome to make my own logo, so I did. I found a black and white image on the web under a search for owl icons that were free to use. I added the color and the glasses. I added the broken glasses that are held together with a band aid as a salute to the old Nerd steryotype.
